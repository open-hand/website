# 个人中心

<h2 id="1">介绍</h2>

个人中心是用户管理个人设置的模块。在个人中心中，您可以管理您的个人信息，包括头像、用户名、邮箱，修改您的个性化设置，包括语言、时区，修改您的平台密码，查看您的组织信息、项目信息，管理授权。

- 在界面的“右上角圆形图标”进入`个人中心`界面。

<h2 id="1">功能</h2>

- [**个人信息**](../person/information) 用户个人信息的管理。

- [**修改密码**](../person/secret_change) 非LDAP用户可以修改自己的登录密码。
