## 介绍
  
本页面介绍了组织设置，组织下的项目、用户、客户端的管理需进入到一个特定的组织进行组织层的操作。

- 在项目层界面中选择一个“组织”，进入“组织设置”界面。

- 需要了解如何[创建组织](..//platform/tenant)。
